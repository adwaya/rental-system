# 🏠 Smart Rental System

A modern **property listing platform** inspired by PropertyPal, built with **Django** and designed for scalability, clarity, and clean UX.

This project is an MVP that allows users to:
- Browse properties for rent or sale
- Filter by location, type, and price
- View detailed property pages
- Manage listings via Django Admin

---

## ✨ Features

### Public
- Property listings grid (modern card UI)
- Search by location
- Filter by rent / sale
- Filter by max price
- Responsive, mobile-friendly design
- Property detail page with optional map tile

### Admin
- Secure Django Admin panel
- Create, update, delete properties
- Manage pricing, type, and details

---

## 🛠 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** Django Templates + CSS
- **Database:** SQLite (default, easy to swap)
- **Styling:** Custom CSS (PropertyPal-inspired)
- **Maps:** Google Maps embed (optional per property)
